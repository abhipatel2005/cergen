Certificate Generator

Generate personalized certificates from Excel data and PDF templates

How to use:

  • Upload an Excel file (.xlsx or .xls) with a "name" column containing the names for certificates
  • NEW! Upload a PowerPoint template (.pptx) with placeholders like {{name}} for true text replacement
  • Or upload a PDF template (legacy support with overlay method)
  • Click "Generate Certificates" to create personalized certificates
  • Download individual certificates or view all generated batches
